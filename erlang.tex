For a **single-accumulator** (local) case:

Let

[
T_D = \text{decision/detection time}
]

[
T_G = \text{guess time}
]

[
T_K = \text{kill/omission time}
]

with all three independent. A response is observed if either the Wald decision wins or the guess clock wins. An omission occurs if the kill clock wins.

So the observed response density is

[
p_R(t)
======

\underbrace{f_D(t)S_G(t)S_K(t)}*{\text{decision wins}}
+
\underbrace{f_G(t)S_D(t)S_K(t)}*{\text{guess wins}}
]

and the omission probability is

[
p_O
===

\int_0^\infty f_K(t)S_D(t)S_G(t),dt .
]

Here (f_D) and (S_D) are the Wald density and survivor.

---

## Wald decision process

Let the Wald decision time be

[
T_D = \tau + X
]

where (X) is a one-boundary diffusion first-passage time with threshold (b), drift (\nu), and diffusion coefficient (\sigma).

The Laplace transform is

[
L_D(s)
======

# \mathbb{E}\left[e^{-sT_D}\right]

\exp\left[
-s\tau
+
\frac{b}{\sigma^2}
\left(
\nu-\sqrt{\nu^2+2\sigma^2s}
\right)
\right].
]

This is the little magic key. Anything involving

[
\int_0^\infty e^{-st}S_D(t),dt
]

can be written using (L_D(s)).

Specifically,

[
I_0(s)
======

# \int_0^\infty e^{-st}S_D(t),dt

\frac{1-L_D(s)}{s}.
]

Also,

[
I_1(s)
======

# \int_0^\infty t e^{-st}S_D(t),dt

-I_0'(s),
]

and

[
I_2(s)
======

# \int_0^\infty t^2 e^{-st}S_D(t),dt

I_0''(s).
]

Explicitly,

[
I_1(s)
======

\frac{1-L_D(s)}{s^2}
+
\frac{L_D'(s)}{s},
]

and

[
I_2(s)
======

\frac{2(1-L_D(s))}{s^3}
+
\frac{2L_D'(s)}{s^2}
--------------------

\frac{L_D''(s)}{s}.
]

For the Wald transform, write

[
r(s)=\sqrt{\nu^2+2\sigma^2s}.
]

Then

[
L_D'(s)
=======

L_D(s)
\left(
-\tau-\frac{b}{r(s)}
\right),
]

and

[
L_D''(s)
========

L_D(s)
\left[
\left(
-\tau-\frac{b}{r(s)}
\right)^2
+
\frac{b\sigma^2}{r(s)^3}
\right].
]

So the omission probability can be evaluated analytically once you reduce it to (I_0,I_1,I_2).

# Case 1: Erlang-1 guess and Erlang-1 kill

Erlang-1 is just exponential.

Let

[
T_G \sim \text{Exp}(\lambda_G),
\qquad
T_K \sim \text{Exp}(\lambda_K).
]

Then

[
f_G(t)=\lambda_G e^{-\lambda_G t},
\qquad
S_G(t)=e^{-\lambda_G t},
]

and

[
f_K(t)=\lambda_K e^{-\lambda_K t},
\qquad
S_K(t)=e^{-\lambda_K t}.
]

Define

[
a=\lambda_G+\lambda_K.
]

The response density is

[
p_R(t)
======

f_D(t)e^{-at}
+
\lambda_G e^{-\lambda_G t}S_D(t)e^{-\lambda_K t}.
]

So

[
p_R(t)
======

e^{-at}
\left[
f_D(t)+\lambda_G S_D(t)
\right].
]

The omission density, if omission time were observed, is

[
p_O(t)
======

\lambda_K e^{-at}S_D(t).
]

Usually omission time is not observed, so we integrate:

[
p_O
===

\lambda_K
\int_0^\infty e^{-at}S_D(t),dt.
]

Using (I_0(a)),

[
p_O
===

\lambda_K I_0(a).
]

Therefore

[
\boxed{
p_O
===

\lambda_K
\frac{1-L_D(a)}{a}
}
]

where

[
a=\lambda_G+\lambda_K.
]

The total probability of a genuine decision response is

[
p_D
===

# \int_0^\infty f_D(t)e^{-at},dt

L_D(a).
]

The total probability of a guess response is

[
p_G
===

# \lambda_G I_0(a)

\lambda_G
\frac{1-L_D(a)}{a}.
]

So the three possible outcomes are

[
\boxed{
p_D=L_D(a)
}
]

[
\boxed{
p_G=\lambda_G\frac{1-L_D(a)}{a}
}
]

[
\boxed{
p_O=\lambda_K\frac{1-L_D(a)}{a}
}
]

and they sum to one:

[
p_D+p_G+p_O=1.
]

Very tidy. Erlang-1 is the little glass bead case.

# Case 2: Erlang-2 guess and Erlang-2 kill

Now let both guess and kill be Erlang-2, using the R convention

[
\texttt{rgamma(shape = 2, rate = }\lambda\texttt{)}.
]

So

[
T_G \sim \text{Gamma}(2,\lambda_G),
\qquad
T_K \sim \text{Gamma}(2,\lambda_K).
]

The density and survivor are

[
f_G(t)=\lambda_G^2 t e^{-\lambda_G t},
\qquad
S_G(t)=e^{-\lambda_G t}(1+\lambda_G t),
]

and

[
f_K(t)=\lambda_K^2 t e^{-\lambda_K t},
\qquad
S_K(t)=e^{-\lambda_K t}(1+\lambda_K t).
]

Again define

[
a=\lambda_G+\lambda_K.
]

The response density is

[
p_R(t)
======

f_D(t)S_G(t)S_K(t)
+
f_G(t)S_D(t)S_K(t).
]

Substitute the Erlang-2 forms:

[
p_R(t)
======

f_D(t)
e^{-at}
(1+\lambda_G t)(1+\lambda_K t)
+
\lambda_G^2t e^{-\lambda_G t}
S_D(t)
e^{-\lambda_K t}(1+\lambda_K t).
]

So

[
\boxed{
p_R(t)
======

e^{-at}
\left[
f_D(t)(1+\lambda_G t)(1+\lambda_K t)
+
\lambda_G^2t S_D(t)(1+\lambda_K t)
\right]
}
]

This is still closed-form for observed RTs because it only uses the Wald density and survivor at (t).

Now the omission probability:

[
p_O
===

\int_0^\infty f_K(t)S_D(t)S_G(t),dt.
]

Substitute:

[
p_O
===

\int_0^\infty
\lambda_K^2t e^{-\lambda_K t}
S_D(t)
e^{-\lambda_G t}(1+\lambda_G t)
,dt.
]

Combine exponentials:

[
p_O
===

\lambda_K^2
\int_0^\infty
t e^{-at}(1+\lambda_G t)S_D(t)
,dt.
]

Expand the polynomial:

[
p_O
===

\lambda_K^2
\left[
\int_0^\infty t e^{-at}S_D(t),dt
+
\lambda_G
\int_0^\infty t^2 e^{-at}S_D(t),dt
\right].
]

Using (I_1(a)) and (I_2(a)),

[
\boxed{
p_O
===

\lambda_K^2
\left[
I_1(a)+\lambda_G I_2(a)
\right]
}
]

where

[
I_1(a)
======

\frac{1-L_D(a)}{a^2}
+
\frac{L_D'(a)}{a},
]

and

[
I_2(a)
======

\frac{2(1-L_D(a))}{a^3}
+
\frac{2L_D'(a)}{a^2}
--------------------

\frac{L_D''(a)}{a}.
]

So fully expanded:

[
\boxed{
p_O
===

\lambda_K^2
\left[
\frac{1-L_D(a)}{a^2}
+
\frac{L_D'(a)}{a}
+
\lambda_G
\left(
\frac{2(1-L_D(a))}{a^3}
+
\frac{2L_D'(a)}{a^2}
--------------------

\frac{L_D''(a)}{a}
\right)
\right]
}
]

with

[
a=\lambda_G+\lambda_K.
]

---

## Total probabilities in Erlang-2 case

The probability that the Wald decision wins is

[
p_D
===

\int_0^\infty
f_D(t)
e^{-at}
(1+\lambda_G t)(1+\lambda_K t)
,dt.
]

Expand:

[
(1+\lambda_G t)(1+\lambda_K t)
==============================

1+(\lambda_G+\lambda_K)t+\lambda_G\lambda_Kt^2.
]

So

[
p_D
===

## L_D(a)

(\lambda_G+\lambda_K)L_D'(a)
+
\lambda_G\lambda_K L_D''(a).
]

Thus

[
\boxed{
p_D
===

## L_D(a)

aL_D'(a)
+
\lambda_G\lambda_KL_D''(a)
}
]

The probability that the guess clock wins is symmetric to the kill expression:

[
\boxed{
p_G
===

\lambda_G^2
\left[
I_1(a)+\lambda_K I_2(a)
\right]
}
]

and the omission probability is

[
\boxed{
p_O
===

\lambda_K^2
\left[
I_1(a)+\lambda_G I_2(a)
\right]
}
]

with

[
p_D+p_G+p_O=1.
]

---

## Interpretation

For Erlang-1,

[
h_G(t)=\lambda_G,
\qquad
h_K(t)=\lambda_K.
]

So guess and kill are constant-hazard processes. They can happen immediately, and their urgency does not build over time.

For Erlang-2,

[
h_G(t)=\frac{\lambda_G^2t}{1+\lambda_G t},
\qquad
h_K(t)=\frac{\lambda_K^2t}{1+\lambda_K t}.
]

These start at zero:

[
h_G(0)=0,
\qquad
h_K(0)=0,
]

then rise over time:

[
h_G(t)\to \lambda_G,
\qquad
h_K(t)\to \lambda_K.
]

So (\lambda_G) controls how quickly timed guessing pressure ramps up, and (\lambda_K) controls how quickly omission pressure ramps up.

In likelihood form:

[
\ell(t,\text{response})=\log p_R(t)
]

for observed responses, and

[
\ell(\text{omission})=\log p_O
]

for omitted trials.

That gives you a single-accumulator Wald model with both **timed responding** and **timed omission**, still analytic for Erlang-1 and Erlang-2.
